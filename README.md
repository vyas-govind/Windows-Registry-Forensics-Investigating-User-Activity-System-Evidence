# Windows-Registry-Forensics-Investigating-User-Activity-System-Evidence
A hands-on digital forensics project focused on Windows Registry analysis.
Reconstructed user and system activity using persistent registry artifacts.
Identified executed programs, file access, USB usage, and login behavior.
Correlated timestamps to build a defensible activity timeline.
Demonstrated structured DFIR workflow and artifact interpretation.
Focused on offline registry hive examination in a controlled lab environment.
Showcases forensic reasoning, documentation, and investigative methodology.
